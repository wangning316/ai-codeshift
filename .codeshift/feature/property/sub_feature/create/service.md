# 商品属性项_创建(service)

## service粒度
- 独立的商品属性项_创建service类，ProductPropertyCreateService。

## 方法描述
- 创建商品属性项

## 业务逻辑
1. 传入`创建商品属性项-请求参数(Body)`对象。
2. 参数校验：根据传入的`创建商品属性项-请求参数(Body)-属性项名称`使用`商品属性项_创建(mapper)`的数据库操作对象查询数据库，校验属性项名称是否已存在；若已存在，直接返回数据库中已存在的`商品属性项(entity)-属性项编号`。
3. 数据转换：将传入的`创建商品属性项-请求参数(Body)`，转换为数据库对应的`商品属性项(entity)`数据对象。
4. 数据库操作：使用`商品属性项_创建(mapper)`的数据库操作对象将`商品属性项(entity)`数据对象插入数据库。
5. 方法返回：返回数据库自动生成的商品`商品属性项(entity)-属性项编号`。
